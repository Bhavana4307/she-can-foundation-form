# She Can Foundation Form

This project is a simple web form created as part of the Full Stack Development Internship Task.

## Features

- Name Field
- Email Field
- Message Field
- Submit Button
- Success Message after submission

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. User enters their name, email, and message.
2. User clicks the Submit button.
3. A "Form Submitted Successfully" message is displayed.
4. The form fields are cleared automatically.
